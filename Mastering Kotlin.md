# Kotlin Syllabus

## Week 1: Introduction to Kotlin

*   What is Kotlin? History, Features, Benefits.
*   Kotlin vs. Java
*   Setting up the Development Environment (IntelliJ IDEA, Gradle)
*   Basic Syntax: `main` function, variables, data types (Int, Double, Boolean, String)
*   Comments
*   Basic Input/Output (println)

## Week 2: Variables, Data Types, and Operators

*   Variable Declaration: `val` (immutable), `var` (mutable)
*   Data Types: `Int`, `Long`, `Float`, `Double`, `Boolean`, `Char`, `String`
*   Type Inference
*   Operators: Arithmetic, Relational, Logical, Assignment
*   String Templates
*   Null Safety: Nullable and Non-Nullable Types (`?`, `!!`)
*   Elvis Operator (`?:`)

## Week 3: Control Flow

*   Conditional Statements: `if`, `else if`, `else`
*   `when` Expression (Kotlin's enhanced switch statement)
*   Loops: `for`, `while`, `do-while`
*   Ranges (`..`, `until`, `step`)
*   `break` and `continue` statements

## Week 4: Functions

*   Function Declaration: `fun` keyword
*   Function Parameters and Return Types
*   Default Arguments
*   Named Arguments
*   Single-Expression Functions
*   Extension Functions
*   Infix Functions

## Week 5: Collections

*   Lists: `List`, `MutableList`
*   Sets: `Set`, `MutableSet`
*   Maps: `Map`, `MutableMap`
*   Iterating through Collections
*   Collection Operations: `filter`, `map`, `forEach`, `reduce`, `groupBy`, `sorted`

## Week 6: Object-Oriented Programming (OOP)

*   Classes and Objects
*   Properties and Methods
*   Constructors: Primary and Secondary
*   Inheritance
*   Abstract Classes and Interfaces
*   Visibility Modifiers: `public`, `private`, `protected`, `internal`
*   Data Classes

## Week 7: Advanced OOP

*   Sealed Classes
*   Enum Classes
*   Delegation
*   Companion Objects
*   Object Expressions and Declarations
*   Generics

## Week 8: Functional Programming Concepts

*   Lambda Expressions
*   Higher-Order Functions
*   Closures
*   Function Types
*   Immutability
*   Pure Functions

## Week 9: Coroutines

*   Introduction to Asynchronous Programming
*   Threads vs. Coroutines
*   `suspend` Functions
*   `CoroutineScope` and `CoroutineContext`
*   `launch` and `async` builders
*   `withContext`

## Week 10: Exception Handling

*   `try`, `catch`, `finally` blocks
*   `throw` keyword
*   Custom Exceptions
*   `use` function for resource management

## Week 11: Testing

*   Unit Testing with JUnit and Mockito
*   Writing Testable Code
*   Test-Driven Development (TDD)

## Week 12: Working with APIs

*   Making HTTP Requests with Ktor or Retrofit
*   Parsing JSON data with kotlinx.serialization or Gson
*   Handling API responses

## Week 13: Android Development with Kotlin (Introduction)

*   Setting up Android Studio
*   Creating a basic Android application
*   UI Components: TextView, EditText, Button
*   Layouts: LinearLayout, RelativeLayout, ConstraintLayout
*   Activities and Fragments

## Week 14: Android Development with Kotlin (Continued)

*   Event Handling: Click Listeners
*   Data Binding
*   RecyclerView and Adapters
*   Networking in Android (using Coroutines)
*   Room Persistence Library
