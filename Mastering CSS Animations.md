# CSS Animations Syllabus

**Week 1: Introduction to CSS Animations**

*   What are CSS Animations?
*   Keyframes: The Foundation of Animation
*   `@keyframes` Rule: Defining Animation Sequences
*   `animation-name`: Linking Keyframes to Elements
*   `animation-duration`: Setting the Animation Length

**Week 2: Animation Properties**

*   `animation-timing-function`: Controlling Animation Speed
    *   `linear`, `ease`, `ease-in`, `ease-out`, `ease-in-out`
    *   Custom Cubic Bezier Functions
*   `animation-delay`: Starting Animations After a Delay
*   `animation-iteration-count`: Looping Animations
    *   Numeric Values
    *   `infinite`
*   `animation-direction`: Animation Playback Direction
    *   `normal`, `reverse`, `alternate`, `alternate-reverse`

**Week 3: Advanced Animation Techniques**

*   `animation-fill-mode`: Applying Styles Before/After Animation
    *   `none`, `forwards`, `backwards`, `both`
*   `animation-play-state`: Controlling Animation Playback
    *   `running`, `paused`
*   Shorthand `animation` Property
*   Practical Examples: Simple Animations

**Week 4: Transitions vs. Animations**

*   CSS Transitions: A Quick Review
*   Key Differences Between Transitions and Animations
*   When to Use Transitions vs. Animations
*   Combining Transitions and Animations for Complex Effects

**Week 5: Animating Transformations**

*   CSS Transforms: A Quick Review
    *   `translate`, `rotate`, `scale`, `skew`
*   Animating Transforms with Keyframes
*   Creating Complex Motion with Transform Combinations

**Week 6: Animating Opacity and Colors**

*   Animating Opacity for Fade-In/Fade-Out Effects
*   Animating Background Colors and Text Colors
*   Creating Color Transition Effects

**Week 7: Responsive Animation & Performance**

*   Adapting Animations to Different Screen Sizes
*   Media Queries for Animation Control
*   Performance Considerations: Avoiding Janky Animations
    *   Using `transform` and `opacity` for better performance
    *   Reducing paint/layout recalculations
*   Tools for Measuring Animation Performance

**Week 8: Advanced Animation Projects & Libraries**

*   Building a Complex Animation Project
*   Introduction to CSS Animation Libraries (e.g., Animate.css)
*   Leveraging Libraries for Pre-Built Animations
*   Customizing Library Animations
